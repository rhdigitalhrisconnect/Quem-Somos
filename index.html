NOSSOS PILARES

O que nos motiva neste projeto

Guiados pelo aprendizado prático e pela vontade de fazer a diferença, nosso trabalho se baseia em três pilares fundamentais.

Colaboração

Desenvolvemos esta solução unindo conhecimentos diversos de tecnologia e gestão, simulando um ambiente real de trabalho em equipe.

Inovação

Buscamos criar uma ferramenta tecnológica que traga soluções modernas e eficientes para os desafios enfrentados pelo RH no dia a dia.

Aprendizado

Aplicamos os conceitos aprendidos no curso de Serviços Administrativos do SENAC para transformar um desafio acadêmico em um projeto prático e relevante.

O PROJETO RH DIGITAL

Simplificando o RH com Tecnologia

O RH Digital - HRIS Connect nasceu da necessidade de modernizar processos, reduzir a burocracia e permitir que os profissionais de RH foquem no capital humano.

  • Gestão centralizada de candidatos e funcionários.
  • Ferramentas para otimizar o recrutamento e seleção.
  • Calendário dinâmico para organização de tarefas e entrevistas.
  • Dashboards intuitivos para análise de dados.
  • Interface amigável e acessível.

PERGUNTAS FREQUENTES

Dúvidas sobre o Projeto e a Equipe

Encontre respostas para perguntas comuns sobre o desenvolvimento do RH Digital.

Qual o principal objetivo deste Projeto Integrador?

O objetivo principal é aplicar os conhecimentos adquiridos no curso de Serviços Administrativos do SENAC em um projeto prático, desenvolvendo uma solução tecnológica (RH Digital) que aborda desafios reais da área de Recursos Humanos, como a otimização de processos e a gestão de talentos.

Quem são os integrantes da equipe?

A equipe é composta pelos alunos da Turma 170 ADM do curso de Aprendizagem Profissional de Qualificação em Serviços Administrativos do SENAC. Cada membro contribuiu com suas habilidades e conhecimentos para as diferentes fases do projeto, desde a concepção até o desenvolvimento.

Quais tecnologias foram usadas no desenvolvimento?

O projeto utilizou tecnologias web modernas. Para o Front-End (interface do usuário), foram empregados HTML, CSS e JavaScript. Para o Back-End (lógica do servidor e banco de dados), utilizamos Python com o framework Flask e um banco de dados SQL (como SQLite ou PostgreSQL, dependendo da fase).

O RH Digital já está pronto para uso comercial?

Atualmente, o RH Digital - HRIS Connect é um protótipo funcional desenvolvido como Projeto Integrador. Ele demonstra as funcionalidades chave e o potencial da plataforma. Para uso comercial completo, seriam necessárias fases adicionais de testes, segurança aprofundada e desenvolvimento de mais funcionalidades, além de uma infraestrutura robusta.

Como posso saber mais ou dar feedback?

Ficamos felizes com seu interesse! Você pode entrar em contato através das nossas redes sociais (link no rodapé) ou, se disponível, utilizar a seção de contato da plataforma principal. Seu feedback é muito valioso para nós!
